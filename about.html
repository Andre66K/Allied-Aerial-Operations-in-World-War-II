About

Data

The dataset used for this project was retrieved in June 2020 from Kaggle.com. The dataset comes from THOR an historical data collection project developed by Air Force Lt. Col. Robertson and released to the public by the Department of Defense in 2012. I tried to locate the original source of the data, but the webpage of the Air Force Research Institute where the original database was allegedly released is not working properly as of June 2020. The data, however, can be found on many different online war and historical blogs as well as on data science online communities such as Kaggle and data.world.

Methodology and Data Cleaning

The dataset was downoladed in CSV format and it was cleaned and processed using Python and Excel. The original dataset has over 178k rows and 62 columns; however, as it contains many missing values, the number of columns was cut to 21 columns. In some cases, the missing values were replaced with 0s, as the missing values do not indicate missing values per se but indicate that a specific type of aircraft, or bomb, were not used for the operation. In many other cases the missing values do indicate real missing values, which are often displayed as 'Not Recorded' in the visualizations. The aircrafts' specifications information was collected into a CSV file manually. The CSV file was then merged with the main dataset.
